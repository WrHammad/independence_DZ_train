# independence_DZ_train
قطار الاستقلال: لعبة تفاعلية تعليمية توثق أحداث مظاهرات 11 ديسمبر 1960 الحاسمة في تاريخ الثورة الجزائرية. ينتقل اللاعب في رحلة عبر 4 محطات تاريخية، تبدأ من سياق زيارة ديغول وتنتهي بالانتصار الدبلوماسي في الأمم المتحدة. يتطلب الفوز ترتيب الأحداث الجغرافية والزمنية ومعرفة رموز الثورة بدقة لقيادة القطار نحو محطة الحرية. (من إعداد الكاتب حماد بن عيسى)
